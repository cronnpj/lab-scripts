# MISC Assets

Store reusable non-script assets here for Lab Tools, such as:
- executables (`.exe`)
- images/backgrounds
- helper binaries
- reference files

This folder is part of the deployable `src` tree, so content here can be used by menu options at runtime.

## vmPing Placement

Place vmPing in:

- `src/MISC/vmPing/vmPing.exe`

After placing the executable, launch it from:

- `Main > Windows Client Tools > [16] Launch vmPing (MISC)`
