# VMware Horizon OS Optimization Tool

Place the tool executable in this folder with this file name:

- `VMwareHorizonOSOptimizationTool-x86_64.exe`

Runtime menu path:

- `Main > Windows Client Tools > Utilities > [5] Launch VMware Horizon OS Optimization Tool`

Task behavior:

- Option [1]: launch interactive UI.
- Option [2]: attempt quiet run/install with arguments (default `/S`, editable at runtime).
